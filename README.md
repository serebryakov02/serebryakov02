## About

I build modern C++/Qt desktop applications with CMake-based build systems, careful debugging, and maintainable GUI architecture.

I work full time as a software engineer and use this profile for public projects, learning work, and open-source contributions. Alongside C++/Qt, I am actively expanding into AI engineering with Python, OpenAI APIs, agent workflows, MCP, RAG concepts, and practical automation.

## Focus

- C++ desktop applications with Qt 5/6, Qt Widgets, QML, CMake, STL, Boost, multithreading, IPC, and platform integration.
- Software quality through root-cause analysis, code review, OOP/SOLID design, TDD, GoogleTest, Catch2, CI/CD, and clear architecture.
- AI-enabled tools with Python, OpenAI APIs, OpenAI Agents SDK, MCP servers, agent workflows, CrewAI, Gradio, speech-to-text, OAuth integrations, and lightweight RAG-style flows.
- Practical product work across GUI design, data handling, build systems, automation, and engineering tooling.

- [Algorithms Visualization](https://github.com/serebryakov02/Algorithms_Visualization) - Qt desktop app for sorting, graph, and binary search visualizations with interactive controls.

- [github-portfolio-agent](https://github.com/serebryakov02/github-portfolio-agent) - Chat agent that uses OpenAI tools and the GitHub API to discuss software projects.
